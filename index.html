👋 Setup guide — dismiss once you're done
  1. Create 5 folders in Google Drive named exactly: 1 unit, 2 units, 3 units, 4 units, sold.
  2. Upload your ceramic photos into the right folder. Name each file like blue-bowl.jpg — the filename becomes the piece title on the page.
  3. Share each folder publicly: right-click → Share → "Anyone with the link" → Viewer.
  4. Open each folder in your browser and copy its ID from the URL (the part after /folders/). Paste each one into the CONFIG.folderIds section at the top of this file.
  5. Get a free API key: go to console.cloud.google.com → New project → Enable "Google Drive API" → Credentials → Create API Key. Paste it into CONFIG.apiKey.
  6. Fill in your name and email in CONFIG, save, and open in your browser. Done!
  7. To update stock later: just move a photo between folders (e.g. 3 units → 2 units, or → sold). The site updates automatically on next load.
